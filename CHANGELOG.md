# Changelog

## 0.1.1

- Added `inertia.back(c, fallback?, status?)` for Laravel-style redirect-back behavior.
- Added tests for `back()` with `Referer` and fallback redirects.
- Added GitHub Actions CI and docs deployment workflows.
- Added VitePress blog pages and sitemap generation for the documentation site.
- Fixed GitHub Pages base path for the `huno-inertia` repository.

## 0.1.0

- Initial release of `hono-inertia`.
- Added Inertia page rendering for Hono.
- Added custom string/TSX root view support.
- Added shared props, lazy props, optional props, and always props.
- Added Inertia redirects, external redirects, partial reloads, and asset version mismatch handling.
